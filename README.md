# 🚗 Autoclass API (Back-end)

API RESTful desenvolvida em Django para o gerenciamento de um catálogo de veículos de luxo. Este projeto atua como o motor de dados (back-end), fornecendo uma estrutura robusta, relacional e filtrável para consumo via front-end.

## 🛠️ Tecnologias Utilizadas

* **Linguagem:** Python
* **Framework:** Django / Django REST Framework (DRF)
* **Banco de Dados:** SQLite3 (Ambiente de Desenvolvimento)
* **Mídia:** Pillow (Processamento de imagens)

## ⚙️ Funcionalidades Principais

* **CRUD de Marcas e Carros:** Gerenciamento completo de entidades com relacionamento (1:N).
* **Filtros Avançados:** Busca dinâmica por marca, ano de fabricação, modelo e proprietário.
* **Upload de Imagens:** Suporte nativo para imagens atreladas aos veículos.
* **Painel Administrativo:** Interface Django Admin customizada com filtros e barras de pesquisa.
